# CKA Dump Solutions

This repository contains solutions to the CKA (Certified Kubernetes Administrator) practice questions. The questions used here were taken from a publicly available YouTube video.

> **Disclaimer:** These solutions are meant for **practice and educational purposes only**. They are not affiliated with or endorsed by the CNCF or the official CKA exam.

## Source of Questions

The questions are based on the YouTube video:  
[CKA Practice Questions Video](https://youtu.be/rP-W3Tv3plw?si=JI0fJOL6x5BlVAlG)


## How to Use

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd dump3
   ```

2. Browse the `dump3/` folder to review individual question solutions.

## Notes

* Solutions are intended for **self-study**.
* Please do not share or distribute these solutions as official exam material.
* Always practice hands-on on your own Kubernetes cluster for the best preparation.
